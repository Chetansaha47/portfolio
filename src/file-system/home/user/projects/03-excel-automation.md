



## *Sales Order & Stock Allocation Automation*
## 2024-25

### • Python, Polars, FastAPI,
### • React, Data Engineering
!(/images/excel-automation.png?aspect=1.7777)
Built a multi-stage Python/Polars data processing pipeline for an Excel-based sales order management system — covering dealer/distributor segmentation, credit limit lookups, billing loops, CF flagging, and stock allocation with depot pool splits.

A related Stock Allocation Engine reverse-engineered business logic purely from sample input/output files and was delivered as a strict, dependency-light Polars pipeline. Also built a prompt-driven Excel automation platform (FastAPI + React + Anthropic API) that works like a lightweight ETL tool for non-technical users.
