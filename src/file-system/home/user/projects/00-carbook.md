



## *Carbook*
## 2025-26

### • Java, JSP, Servlets,
### • MySQL, Security Engineering
!(/images/carbook.png?aspect=1.7777)
Carbook is a car rental web platform built with JSP, Servlets and MySQL, featuring a black/white/yellow editorial UI. It includes comprehensive security work: PBKDF2 password hashing, CSRF protection, XSS prevention and role-based access filters for customers and admins.

Currently being migrated from the older javax.servlet APIs to jakarta.servlet for deployment on Tomcat 10.1 — an ongoing project.
